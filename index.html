🐸 Shreksophone

Le seul CDN qui fait swinguer ton site web 🪗

Bienvenue dans le projet le plus inutile, et donc le plus essentiel, de tout GitHub.
Shreksophone est un CDN troll qui transforme n’importe quel bouton ou lien en Shrek dansant sur un solo de saxophone magique ✨. Parfait pour perturber vos collègues, tester la patience de vos amis, ou simplement honorer le marais.

🎥 Vidéo

🧩 Utilisation en CDN

Choisis ton niveau de chaos. Copie, colle, et laisse Shrek envahir ton site 💫
Disponible via le CDN officiel ou directement sur GitHub Pages.

🎯 Version ciblée – shrek.min.js

Affecte uniquement les éléments avec la classe shrek-troll. 

<button class="shrek-troll">Clique pas</button>
<button class="shrek-troll">Moi non plus</button>

<script src="https://shrek.hexanti.fr/dist/shrek.min.js"></script>

💥 Version totale – shrek-all.min.js

Affecte tous les boutons et liens (<button> et <a>). 

<a href="#">Ne clique pas</a>
<button>Moi non plus</button>

<script src="https://shrek.hexanti.fr/dist/shrek-all.min.js"></script>
